## Daily Posting Protocol (25 min)

1. Open the next post file from the current series folder
2. Copy the **7-line body** (lines 1-13) to LinkedIn
3. Copy the **hashtag line** (after ---) as-is
4. Add the **💡 #thrive footer** as inline text below the hashtags
5. Post
6. Engage with comments for 10 min within the first hour
