Nyk has shipped 32 products. Built 155 repositories. Earned 2,800 GitHub stars on a single project. His thesis in one sentence: "The harness is the product. The model never was."

His flagship, Mission Control, is an open-source dashboard for managing agent fleets. 32 panels. 101 API routes. 39 database migrations. Zero external dependencies. Everything runs on SQLite.

The interesting part is not the technology. It is the philosophy. Quality gates block task completion without sign-off. Memory compounds across sessions. Skills sync bidirectionally between disk and database.

Every feature is about governance, not capability.

The world is building smarter models. Nyk is building smarter harnesses. He might be right.
